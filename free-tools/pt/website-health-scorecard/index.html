      Perguntas Frequentes

      O que é uma pontuação de saúde do site? ▾
      Uma pontuação de saúde do site é uma avaliação de 0 a 100 que mede o desempenho do seu site em cinco áreas críticas: velocidade de carregamento, compatibilidade mobile, segurança SSL, completude das meta tags e presença no Google Business. Uma pontuação mais alta significa que o site tem mais probabilidade de classificar bem nas pesquisas e converter visitantes em clientes.
      Como verifico se o meu site é mobile-friendly? ▾
      A nossa ferramenta simula a deteção de viewport mobile e verifica padrões de design responsivo. Para um teste real, abra o site num smartphone ou use a emulação de dispositivo do Chrome DevTools. Se o texto for demasiado pequeno, os botões difíceis de tocar, ou for necessário scroll horizontal, o site precisa de melhorias responsivas.
      Porque é que o meu site precisa de um certificado SSL? ▾
      Um certificado SSL (HTTPS) encripta dados entre os visitantes e o servidor. Sem ele, os browsers mostram avisos "Não Seguro", que afastam visitantes. O Google também usa HTTPS como sinal de classificação, por isso a falta dele prejudica a visibilidade nas pesquisas. Certificados gratuitos estão disponíveis via Let's Encrypt.
      Que meta tags o meu site precisa para SEO? ▾
      Cada página precisa de uma tag <title> única (menos de 60 caracteres), uma meta description (150–160 caracteres) e uma meta tag viewport para mobile. Estas ajudam os motores de busca a entender o conteúdo e influenciam como as páginas aparecem nos resultados de pesquisa. Tags em falta ou duplicadas prejudicam as classificações.
