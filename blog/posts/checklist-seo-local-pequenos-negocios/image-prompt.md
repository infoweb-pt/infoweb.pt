# Image generation prompt

Criar uma imagem hero 1200x630 para um artigo InfoWeb com o título “Checklist de SEO Local para Pequenos Negócios Receberem Mais Contactos”.

Estilo: ilustração editorial moderna, estética premium de website/agência, fundo azul-marinho/slate escuro, detalhes dourados subtis (#d7b46a), visual prático e claro.

Cena: pequeno negócio local representado de forma abstrata, com pin de mapa, cartão de pesquisa genérico estilo Google (sem logótipo Google), ícone de telefone/contacto, estrelas de avaliação, checklist e uma janela simples de website. Mostrar a ideia de descoberta local a gerar chamadas/contactos.

Cores/marca: paleta navy/dourado da InfoWeb, formas geométricas limpas, sensação amigável de pequeno negócio português, sem ficar demasiado técnico. Usar apenas UI e ícones genéricos.

Composição: layout 16:9 para Open Graph, visual central forte com checklist/mapa, espaço livre no canto superior esquerdo para eventual título, legível em preview pequeno.

Evitar: logótipo Google, screenshots de Maps, nomes reais de negócios, dados falsos de clientes, dashboards confusos, texto pequeno ilegível, marcas de água, tipografia distorcida.

Output: `featured.jpg` ou `featured.webp` em 1200x630, otimizado abaixo de 250KB se possível.
