# Image generation prompt

Criar uma imagem hero 1200x630 para um artigo do blog InfoWeb com o título “Como Escolher o Domínio Perfeito para o Seu Negócio”.

Estilo: ilustração editorial moderna, estética premium de website/agência, fundo azul-marinho escuro alinhado com a marca InfoWeb, detalhes subtis em dourado, composição limpa, profissional e acessível.

Cena: dono de pequeno negócio a usar um portátil e a comparar opções de domínio. Mostrar elementos abstratos de barra de endereço/browser, um cartão “.pt”, ícones simples de globo/DNS e pequenas notas de identidade da marca. A imagem deve servir para qualquer pequeno negócio português, sem ficar demasiado técnica.

Cores/marca: usar tons deep slate/navy, dourado suave (#d7b46a), acentos brancos se necessário, sem logótipos de terceiros, sem domínios reais, sem interfaces confusas.

Composição: deixar espaço livre seguro à esquerda ou no canto superior esquerdo para eventual texto/título. Evitar texto pequeno ilegível. Layout 16:9 otimizado para Open Graph.

Não incluir: rostos em grande plano, aspeto de stock photo, screenshots de registadores reais, logótipos DNS.PT/GoDaddy/Namecheap, marcas de água, texto distorcido.

Output: `featured.png` em 1200x630, otimizado para web com menos de 250KB se possível.
