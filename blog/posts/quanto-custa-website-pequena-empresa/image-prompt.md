# Image generation prompt

Criar uma imagem hero 1200x630 para um artigo InfoWeb com o título “Quanto Custa um Website para Pequenas Empresas em 2026?” (o ano pode atualizar dinamicamente no site).

Estilo: ilustração editorial moderna, estética premium para pequenos negócios, fundo azul-marinho/slate escuro, detalhes dourados subtis (#d7b46a), composição limpa e de alto contraste.

Cena: dono de pequeno negócio a rever um orçamento simples de website num portátil. Incluir cartões abstratos com conceitos genéricos como “domínio”, “hosting”, “design”, “manutenção” e “SEO”, sem valores exatos. Mostrar uma janela de browser simples e elementos de calculadora/orçamento.

Cores/marca: profissional, claro e confiável, sem aspeto de stock photo. Usar paleta navy/dourado alinhada com InfoWeb. Sem logótipos de terceiros, sem screenshots de marcas, sem preços reais.

Composição: layout 16:9 para Open Graph, deixar espaço livre no canto superior esquerdo para possível título, visual central legível em preview pequeno.

Evitar: tabelas ilegíveis, UI confusa, imagens exageradas de dinheiro, montes de euros/dólares, marcas de água, texto distorcido.

Output: `featured.png` em 1200x630, otimizado abaixo de 250KB se possível.
