# Image generation prompt

Criar uma imagem hero 1200x630 para um artigo InfoWeb com o título “Ainda Precisa de um Website se o Seu Negócio Já Tem Instagram?”.

Estilo: ilustração editorial moderna, estética premium para pequenos negócios, fundo azul-marinho/slate escuro, detalhes dourados subtis (#d7b46a), visual limpo e confiável.

Cena: composição dividida com um smartphone a mostrar um feed social abstrato de um lado e um website profissional num portátil do outro. Mostrar uma passagem simples de descoberta nas redes sociais para conversão no website. Incluir ícones subtis de marcação, serviços, avaliações e pesquisa Google.

Cores/marca: usar paleta navy/dourado da InfoWeb. O lado do website deve parecer estável e controlado; o lado social deve parecer ativo mas dependente de plataforma. Não usar logótipo do Instagram nem logótipos de terceiros.

Composição: layout 16:9 para Open Graph, hierarquia clara, espaço livre no canto superior esquerdo para eventual título, legível em preview pequeno.

Evitar: UI real do Instagram, logótipos Meta/Instagram, screenshots, estética de influencer, notificações com texto legível, marcas de água, texto distorcido.

Output: `featured.jpg` ou `featured.webp` em 1200x630, otimizado abaixo de 250KB se possível.
