# Website Maintenance Checklist for Small Businesses

A website is not a one-time project. It is infrastructure — like heating or inventory. Skip maintenance and you get broken forms, outdated hours, security scares, and SEO decay.

Use this checklist monthly and quarterly. You do not need a technical team if tasks are documented — or if maintenance is included in a managed plan.

## Monthly tasks (30–60 minutes)

### Content freshness

- [ ] Opening hours and holiday closures correct
- [ ] Prices, menus, or service lists updated
- [ ] Team photos and “about” copy still accurate
- [ ] Promotions removed when expired

### Forms and contact paths

- [ ] Submit a test on contact/booking forms
- [ ] Confirm emails arrive in the right inbox (not spam)
- [ ] Click-to-call and WhatsApp links work on mobile

### Analytics glance

- [ ] Check traffic trend vs last month
- [ ] Note top pages and top contact sources
- [ ] Confirm tracking still fires (one test conversion)

## Quarterly tasks (deeper pass)

### Backups

- [ ] Verify automated backups exist
- [ ] Know how to restore (ask provider once)
- [ ] Store critical copy/images separately if possible

### SSL and domain

- [ ] Padlock shows in browser
- [ ] Certificate not near expiry
- [ ] Domain renewal date on calendar with auto-renew on

### Software updates

- [ ] CMS, plugins, or theme updates applied safely
- [ ] Remove unused plugins and old admin accounts
- [ ] Change passwords if staff left

### Performance

- [ ] Test homepage speed on mobile data
- [ ] Compress oversized images added recently
- [ ] Fix broken links reported in Search Console

### SEO basics

- [ ] Titles and descriptions still match offers
- [ ] Google Business Profile links to correct URL
- [ ] New services have pages (not only menu PDFs)

### Security

- [ ] 2FA enabled on hosting and registrar
- [ ] No suspicious admin users
- [ ] Spam form submissions monitored

## What to avoid

- “We’ll update it when someone complains” — complaints mean lost leads already happened
- Editing live site without backup before big changes
- Letting an ex-employee keep admin access
- Ignoring failed form deliverability until a customer says they tried three times

## Managed maintenance vs DIY memory

| DIY | Managed plan |
|-----|----------------|
| You remember the checklist | Provider runs updates and monitoring |
| Risk of skipped months when busy | Consistent cadence |
| Cheaper until something breaks | Predictable cost, faster recovery |

For owners who already juggle operations, finance, and staff, **forgetting maintenance is the default** — not a personal failure.

## Bottom line

Maintenance protects revenue. Backups, SSL, updates, accurate content, working forms, and light SEO checks keep a small business website trustworthy. Do it on a schedule — or pay someone to own the schedule.
